── Quick Start ──────────────────────────────────────────────────

1. Verify the operator is running:

   kubectl get pods -n {{ include "zelyo-operator.namespace" . }} -l {{ include "zelyo-operator.selectorLabels" . | replace ": " "=" | replace "\n" "," }}

2. Create your first SecurityPolicy:

   cat <<EOF | kubectl apply -f -
   apiVersion: zelyo.ai/v1alpha1
   kind: SecurityPolicy
   metadata:
     name: baseline
     namespace: {{ include "zelyo-operator.namespace" . }}
   spec:
     severity: medium
     rules:
       - name: security-context
         type: container-security-context
         enforce: true
       - name: resource-limits
         type: resource-limits
         enforce: true
   EOF

3. Check for violations:

   kubectl get securitypolicies -n {{ include "zelyo-operator.namespace" . }}
